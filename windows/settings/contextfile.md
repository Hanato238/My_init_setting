## AIエージェント振り分け

タスクの種別に応じて Claude Code と Gemini を使い分ける。

### Claude Code（自分）が処理する
- コードの作成・編集・デバッグ・レビュー
- git 操作、ファイル管理、スクリプト実行
- CI/CD、インフラ構成、設定ファイルの変更
- MCP サーバやツールの設定・管理
- README・ドキュメント・コミットメッセージなどファイル編集を伴う文章作成

### Gemini に委譲する（`mcp__gemini-cli__ask-gemini`）
- 調査・リサーチ・情報収集
- 文章生成・翻訳・要約・校正（ファイル編集なし）
- ブレスト・アイデア出し → `mcp__gemini-cli__brainstorm`
- 一般的な Q&A、非コード作業全般

### 判断基準
- ファイル変更やコマンド実行を伴う → **Claude Code**
- 知識・文章・発想を求める → **Gemini**
- 混在する場合: コード部分は Claude Code が処理し、調査部分は Gemini に投げる

## ワークフロー

- **中間成果:** ブレストや構成案は `ideas/[YYYYMMDD]-[topic].md` に即座に保存。
- **進捗報告:** 複数のURLをフェッチする場合、各完了ごとに報告を行う。
- **長時間タスク:** ステップを分割し、各ステップ完了後にファイルを保存。
- **説明:** 必ず具体例を含める。
